Intégration Discord — Webhook

Configuration pour importer automatiquement les tickets Discord

URL du Webhook

https://[votre-site].netlify.app/.netlify/functions/webhook

Configurez votre bot Discord pour envoyer une requête POST à cette URL à chaque ouverture de ticket.

Headers requis

Content-Type: application/json X-Webhook-Secret: [votre-secret-dans-netlify-env]

Format JSON attendu

{ "ticket_id": "TVG-2026-001", "type": "plainte", "plaignant": { "nom": "Jean Dupont", "discord_id": "123456789" }, "defendeur": { "nom": "Marc Martin", "discord_id": "987654321" }, "faits": "Description des faits…", "infractions": ["Vol à main armée", "Délit de fuite"], "preuves": ["Capture MDT", "Témoignage"], "channel_id": "channel_discord_id", "guild_id": "guild_discord_id", "created_at": "2026-05-24T10:00:00Z" }

Variables d'environnement Netlify

Dans Netlify → Site configuration → Environment variables, ajoutez :

WEBHOOK_SECRET = [secret-de-votre-choix] FIREBASE_PROJECT_ID = [votre-project-id] FIREBASE_CLIENT_EMAIL = [email-compte-service] FIREBASE_PRIVATE_KEY = [clé-privée-compte-service]
